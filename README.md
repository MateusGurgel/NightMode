## Run Locally

Clone the project

```bash
  git clone https://github.com/MateusGurgel/NightMode
```

Go to the project directory

```bash
  cd NightMode
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open your browser to see the result!
